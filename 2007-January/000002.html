Hi,

A complete overview of the surface modelling functions
in Varkon wold be very comprehensive. I must ask you to
consult the docs and sources for that.

In very short, Varkon supports many kind of curve and surface
representations, probably more than any other system.
NURBS is one of them. Exact analytic offset is another. All Catia
polynomials, Conic Lofting, Rational Cubic and so on. See
DB.h and surdef.h

Varkon supports surface/line, surface/plane but not surface/surface
intersects.

Trimmed surfaces are being implemented at this moment. See the
release notes on the website. Planned release before summer-07.

Hopes this helps, please come back with more specific questions.

Regards, Johan
