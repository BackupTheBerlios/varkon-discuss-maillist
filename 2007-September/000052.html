Hi,

As you all know the Varkon project has moved to SourceForge (SF).

Bookmark: http://sourceforge.net/projects/varkon

The Varkon project website on SF includes functionality to
download packaged official Varkon distributions but
also a source code repository administred by the
Subversion system, SVN.

With SVN you can download the latest fixes and
developments from the source code repository. To
see what has been done, check the news on the Varkon
project site, check the SVN log and also check the
bug tracker for closed bug reports.

Next packaged official release of Varkon will be Varkon 2.0
and the main topic for 2.0 will be to make the system easier to
learn and easier to use. 1.19B was a first step in that direction
but there is much much more to do.

It would be very encouraging for the development team to get
your input into the planning of Varkon 2.0. Please use the forum
on the Varkon project site on SF or add a feature request using
the tracker. You can also use this mailing list to get other
peoples opinions on possible enhancements.

Remember, the main focus now is user friendliness ! How do
we transform Varkon into a system that can beat commersial
competitors ?

Regards, Johan

The Varkon development team
