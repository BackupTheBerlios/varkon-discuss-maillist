Hello Varkon users!

Apparently I just made an instant clean build of Varkon-1.19E on Linux Mint 13 Maya (which is AFAIK derived from Ubuntu 12.04).

Since for the inexperienced user solving the dependencies may be a bit difficult, I made a list of all packages (for Ubuntu 12.04) needed for running and building the latest Varkon release.
I also added the website-URLs of the original authors of the software, in case you want to build it on some different distribution.

In case there are any errors in this list, please drop a line.


- libx11-6 => libx11-dev
  libxpm4 => libxpm-dev
  libxext6 => libxext-dev
  libxdamage1 => libxdamage-dev
  libxfixes3 => libxfixes-dev
  libx11-xcb1 => libx11-xcb-dev
  libxxf86vm1 => libxxf86vm-dev
  libxau6 => libxau-dev
  libxdmcp6 => libxdmcp-dev
http://www.x.org
- libgl1-mesa-glx => libgl1-mesa-dev
  libglu1-mesa => libglu1-mesa-dev
  libglapi-mesa
http://mesa3d.sourceforge.net/
- libtiff4 => libtiff4-dev
http://libtiff.maptools.org/
- libc6 => libc6-dev
http://www.eglibc.org/home
- libxcb1 => libxcb1-dev
  libxcb-glx0 => libxcb-glx0-dev
http://xcb.freedesktop.org/
- libdrm2 => libdrm-dev
http://dri.freedesktop.org/wiki/
- libstdc++6 => libstdc++6-4.6-dev
  libgcc1
http://gcc.gnu.org/
- libjpeg-turbo8 => libjpeg-turbo8-dev
http://libjpeg-turbo.virtualgl.org/
http://www.ijg.org/
- zlib1g => zlib1g-dev
http://zlib.net/
- m4
http://www.gnu.org/software/m4/


Best regards!
