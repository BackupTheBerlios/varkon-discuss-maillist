Hi all,

Thanks to Ruven Gottleib and others who have helped
and encouraged us we have now finally set up the
Varkon system on SourceForge. At the same time
we are releasing Varkon version 1.19B for Unix/Linux.

SourceForge hosts the worlds largest collection
of free software and includes a lot of functionality
to faciliate development, distribution and maintenance
of software.

This means that:

1. The Varkon homepage moves from Orebro university
   to http://varkon.sourceforge.net Please update
   your links. The old website on the university server
   will have a link to the new site for a while but sooner
   or later it will dissapear.

2. There is also a project website for the Varkon project.
   http://sourceforge.net/projects/varkon Please bookmark
   this link as well.

The Varkon project website includes functions for downloading
different versions of the system, functions to report bugs
or request new features, a newsletter and also a source code
repository based on the Subversion system (SVN). Please have
a look and also download the new version of Varkon, 1.19B.

We hope that you will enjoy Varkon 1.19B and also the new
functionality offered through the SourceForge website.

Finally a word about this list. The Sourceforge website
offers a mailing list service similar to this one on the
Berlios server. For the time being we will stay with Berlios
but it is likley that we will move also the mailing list to
Sourceforge in the future. We will keep you informed.

Please visit http://sourceforge.net/projects/varkon and give
us feedback or try the feature request tracking function.

Regards,

The CAD research team
Department of Technology
Orebro university
Sweden
