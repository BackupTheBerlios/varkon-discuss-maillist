Hi,

We are now close to the release of Varkon 1.19
for Linux/Unix. 1.19 includes a major update
of the user interface and (limited) support for
trimmed surfaces. See the release notes on the
Varkon website at:

http://www.tech.oru.se/cad/varkon

Ruven Gottleib, the maintaner of varkon.org, created
a site for Varkon on http://www.sourceforge.net earlier
this year and the plan is to release Varkon version 1.19
through this site instead of the Örebro university
website.

Finally I would also like to ask all of you to keep
your eyes open for nice icons that can be used with the
Varkon user interface. Please visit the forum on:

http://www.varkon.org

to see the contributions so far. I posted some pictures
there today and also details of what is still missing.

Regards, Johan
