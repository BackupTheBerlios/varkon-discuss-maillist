hello johan,

i've finally found a possible solution to the missing/hiding menu window.

your compatriot, from irc://irc.gimp.org/#gimp gave me the solution they are
using for gimp, which loads in a way similar to varkon. the menu window should
be hinted to be a utility window as follows: gtk_window_set_type_hint (window,
GDK_WINDOW_TYPE_HINT_UTILITY);
the source for gtk_window_set_type_hint is at
http://git.gnome.org/browse/gtk+/tree/gdk/x11/gdkwindow-x11.c#n1946.

best regards,
alex
