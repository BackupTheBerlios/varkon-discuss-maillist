Sören has given you an answer regarding the creation of closed surfaces. I would like also to comment the 0.1 units minimum distance requirement.

 

Almost all geometry calculation functions are based on tolerances. These tolerances must be set with respect to the size of the objects. Varkon is programmed and tested for parts in aircrafts, cars, etc. that are defined in millimeter. If Varkon is used for very small objects, or if "normal-sized" objects are defined in meter, then tolerances could be changed, but this is NOT recommended. It is much safer to scale objects or work in millimeter. For more information please refer to:   

 

http://www.ViewsOnCad.ch/VarkonTutorial/tolerances.htm
