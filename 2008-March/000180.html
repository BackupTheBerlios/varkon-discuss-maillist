> No, there is currently no such function. But as we now have  
> implemented the representation of trimmed surfaces in varkon it  
> would be possible to also provide this. I will give it a thought.
>
> Regarding the use and creation of trimmed surfaces there is still a  
> lot neeed in the system i think.
> For example surface/surface intersection etc.
>
> But to convert a planar closed curve to a planar trimmed surface I  
> think is less problematic.
>
>
>
> Such a function may include the following internal steps (from a  
> developer point of view):
>
> 1) verify that the input curve is closed and planar.
>
> 2) create a planar surface (one patch) that include the curve.
>
> 3) Find the transforamtion (incl. scaling)  that would map the curve  
> to the uv-space of the surface
>
> 4) Aply the transforamtion and create the uv-curve
>
> 5) Create the trimmed surface
>
> It is probably possible to implement a test version of this in mbs- 
> code.
>
>
