Hi,

Varkon 1.19C is now released on the SourceForge web site:

http://sourceforge.net/projects/varkon

1.19C is available as a source code package for Linux,
MacOSX and FreeBSD. There is also a prebuilt binary for Linux.

Both distributions include the full documentation which
is also available through the Varkon web site:

http://varkon.sourceforge.net/

Please read the releasenotes before updating. Changes
have been made that may require your attention.

http://varkon.sourceforge.net/man_1.19C/relnotes/index.htm

----------

The main goal for 1.19C has been to make Varkon easier to
install, learn and use. Many of you have contributed with
very important discussions and feedback. There is still a
lot left to do however and we would really like to encourage
you to participate in the discussion on this list or submit
bugs or feature requests to the Varkon website on SourceForge.

A main issue for the future is the Microsoft Windows version
of Varkon. It would be very nice not to have to support two
verions of Varkon at the same time (X11 and WIN32) but we
 still can't find an open source implementation of the full
X11 windowing system for Microsoft Windows. There are X
servers like MINGW but we also need the includes and libs
to be able to build the client (xvarkon). Any ideas would be
greatly appreciated. We know that commersial versions exist.

Meanwhile we will continue the development towards Varkon 2.0
with regular updates of the SVN source repository. Please
give us feedback and help us to go in the right direction !
