Hi,

I've recently been granted approval to host the Varkon and Varkon.Org 
sites on SourceForge.

I haven't uploaded any code to the Varkon site yet, as I'm waiting for 
input from the Varkon developers defining how they want things set up.

-So, developers, send me your input-

I also plan to make Dr. Kjellander an administrator of both Varkon 
sites, so that control of these sites is not solely in my hands. 
Administrator status will give him, and anyone else he chooses, full 
control -equal to my own- over the sites.


http://sourceforge.net/projects/varkon/

The Varkon site will act as a mirror of the official Varkon site, and an 
archive of the Varkon sources. It will also provide the various 
facilities for bug-tracking, feature-requests, support, etc.


http://sourceforge.net/projects/varkon-org/

Likewise, the Varkon.Org site will act as a repository of all (open 
source) code on the Varkon.Org site. As packages are updated, older 
packages will be available only on the SourceForge site.

The SourceForge site will insure the availability of code made available 
to Varkon.Org, so if you want your favorite Varkon stuff put where 
people can get it, you can be assured they'll always find it here.

N.B.: I can host code with any license on the Varkon.Org site, but I can 
only host code with an OSI-approved license on the SourceForge mirror.

I welcome any suggestions, constructive criticism, or help you can 
provide.  I hope these sites prove useful to the Varkon community.

Cheers,
Ruven

PS: Please consider sharing your Varkon code with Varkon.Org. The more 
we have to offer, the more useful Varkon will be to more people.
