Varkon 1.19 is almost done. One of the last
things I would like to do is to replace the
text buttons in the toolbar with icons.

There are a lot of free icons available for
download on the internet but so far I have
not found anything I really like.

It's only a matter of 10-20 icons so we
could create them ourselves. That would also
solve any copyright problems. If anyone is
interested in helping us in any way, please
drop me a line as soon as possible.

We can start with icons for Zoom, Autozoom,
Scale, Pan, Perspective, Previous view, View,
Grid, Levels and Print. I plan to put them at
the top of the graphics window, side by side
so they should all have the same height, something
in the range of 20-30 pixels and XPM format.
