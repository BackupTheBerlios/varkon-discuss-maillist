Regarding Icons

It seems to me that (as Esben-Stien proposes), using svg and also take the tango project guidelines into consideration will be a good idea for the final design of the icons.

Inkscape is a nice tool to create svg illustrations.

However, when in the end the icons will be used as buttons in varkon the xpm format will be used. But it is still a good idea to use the svg format as "originals", a script (using imagemagick / convert) can then generate the buttons of desired size and background color. Such a script and also the svg originals should be included in the varkon distribution.

As Johan writes we could first discuss and agree what symbols to use. For this discussion any simple sketches in any format will do. Later when the final design is to be done the guiding from the tango project will be useful.
