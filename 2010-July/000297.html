Hello CY

Varkon is a parametric evaluator system which means that intersection 
functions should work for all implemented parametric surfaces and curves 
including NURBS. But for every new type of surface I implemented I had 
to do some modifications of the intersection functions.

The surface-curve intersection function should work also for NURBS 
surfaces and curves, but I have never tested.

I never finished the surface-surface intersection function and I don't 
remember how far I came. Another person has now started to work with 
this function (with a new algorithm) and hopefully he will be successful.
