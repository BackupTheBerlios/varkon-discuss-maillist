I'm new on the list, and have'nt workt with varkon yet. But I have a
working copy of Varkon 1.19B on my PowerBook G4 (arch: powerpc).
Mac os X (10..4.11).

Johan Kellander told me to write here about my installation.
Maybe this is'nt the nicest way to go, but it did work for me.

If you don't have, you need to install Xcode and X11.

download and unpack Varkon

download and unpack Mesa3d - MesaLib
from http://sourceforge.net/projects/mesa3d/

copy Mesa-7.0.2/include/GL to Varkon_1.19B/sources/

copy /usr/include/malloc/malloc.h to /usr/include/

now you should be able to build with varkons standard makefile.

I will try to do a port for macports, that should make it easier for mac users.
