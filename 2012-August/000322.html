Glenn,
 
Unfortunately the 64 bit version of varkon is not 100% compatible with
the 32 bit. MBS programs and text files are compatible , but all binary
files, MBO, RES, JOB are not compatible. This is because in the 64 bit
version of Varkon all integers are stored as 64 bit integers.
