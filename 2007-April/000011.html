Since the user interface is being updated, is there any possibility of 
incorporating a 'status line' at the bottom of the user interface window 
in a future upgrade? The suggested change would provide a 'status line' 
to display warnings, error messages, and also it could provide the user 
with a place to enter command aliases as an alternate to menu entries. 
QCAD and AutoCAD both have a 'status line' similar to what I describe. 
Over my years of CAD use I found direct command entry to be very 
convenient since you can enter short commands with your left hand and 
use the mouse with your right hand. AutoCAD lets the user customize his 
own command aliases.

John Hughes
