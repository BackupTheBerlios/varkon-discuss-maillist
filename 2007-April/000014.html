Esben,

Interactive editing of curves is more or less not implemented in Varkon.
As you say, Edit->Entity exists but that's all. A reason why this 
functionality
is missing is that it is a very big job ti implement in a procedural 
parametric
system like Varkon. I actually don't know anyone else who has either.

Varkon supports a large number of different curve representations and many
ways to create each one of them as well as merging them with each other.
Editing control points is only one. We have conics, offsets, NURBS, Cubic
splines, UV-curves etc. etc.

When 1.19 is released (with a new GUI) it is definitely time to take a look
at the interactive create- and edit-functions.

Regards, Johan
