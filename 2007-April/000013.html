John,

A status/message and command line is an interesting possibility.
It would be rather easy to reserve some space at the bottom of the
main graphics window for such a thing. We already display messages
in the lower left corner of the main graphics window but these pop
up and disappear again to leave as much space in the graphics window
as possible for modeling. I suppose a status line(s) must be configurable
in size somehow ? It should also be possible to scroll back I suppose to
get the history ?

What kind of commands would you like to enter ? MBS-statements ? or
maybe define them yourself as MBS MACRO modules. A command would
then simply be the name of a MACRO module. Issuing the command would
call the MACRO module. This function is already available in the file menu.

Regards, Johan
