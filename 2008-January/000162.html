David Keyes has submitted some modules that extend Varkon's capabilities.

There are modules that expand the dimensioning system to include
imperial units, among other things; there's a module which allows the
cloning of entities, and there are some miscellaneous math and string
functions. The code is licensed under the GPL.

There is also an html manual page for each function or module.

See them on the Examples page.
