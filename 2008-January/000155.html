Matthias,

The actual execution of the code should not need
to be slower but the increased size will not permit
the same number of modules to exist in memory as if there
were no comments. This may force modules to be loaded
from disc which would otherwise be cached in memory and
this may slow down execution in some cases.

Compilation/decompilation will be a little slower.

Personally I don't think any of this is a problem today.

Johan
